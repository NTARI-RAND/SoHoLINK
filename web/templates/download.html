SoHoLINK Agent

Run the SoHoLINK agent on a Windows device to contribute compute capacity to the network. The agent runs in the background as a Windows service and registers your device after first launch.

Download for Windows →

Windows 10 or later · ~16 MB · MSI installer

After installation
  1. Run the downloaded MSI installer.
  2. When prompted, paste your registration token. Sign in to your dashboard to generate one.
  3. The agent registers automatically and begins looking for jobs.
Code Signing

SoHoLINK participates in the SignPath Foundation open-source code-signing program. Once verification is complete, Windows installers will be digitally signed by SignPath Foundation, with signing services provided by SignPath.io.
